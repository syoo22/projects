# TIL Blog Validation Automation

## 📌 프로젝트 개요
TIL 챌린지 참여자의 블로그 URL을 자동 검수하여 
커피쿠폰 지급 대상자를 판별하는 Google Apps Script 기반 자동화 시스템

---

## 🚨 도입 배경

기존 방식은 300건이 넘는 블로그 URL을 일일히 수기로 접속하여  
제목과 해시태그를 확인 후 표시하는 형태로 운영되었음.

월 2~3시간 이상 소요되는 비효율적인 구조를 개선하기 위해 자동화 시스템을 구축함.

---

## ⚙️ 동작 흐름

1. A열의 블로그 링크를 순차적으로 읽음
2. 해당 URL에 접속
3. 플랫폼별 HTML 구조에 맞춰 제목/해시태그 영역 추출
4. '멋쟁이사자처럼' 키워드 포함 여부 확인
5. F/G열에 O/X 표시
6. F와 G가 모두 O이면 H열에 O, 아니면 X 표시

---

## 📊 스프레드시트 컬럼 구조

| 열 | 내용 |
|----|------|
| A | 블로그 URL |
| F | 제목 키워드 포함 여부 |
| G | 해시태그 키워드 포함 여부 |
| H | 커피쿠폰 지급 대상 여부 |

---

## 🛠 사용 기술
- Google Apps Script
- Google Spreadsheet
- HTML 파싱 기반 키워드 검증
